# 为什么用使用框架？

框架可以说是对所属的语言，在工程化过程中最好的诠释和理解。所以我们要跟随社区中的大神的思维，看看这些大神们是如何思考！如何理解！

# 社区

[laravel@github](http://github.com/laravel/laravel)

忽悠你改laravel阵营的理由：
- 大神领头开发
- 社区文档性健全、布道者多
- 定期更新、安全性升级
- 商业公司支援
